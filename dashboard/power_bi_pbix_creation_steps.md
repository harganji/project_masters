# Power BI PBIX Creation Steps

Power BI `.pbix` files must be saved from Power BI Desktop. This repository includes the dashboard data and visual design needed to create the PBIX quickly.

## Files to Use

```text
dashboard/car_company_dashboard_data.csv
dashboard/car_company_dashboard.png
docs/visualization_guide_powerbi_tableau.md
```

## Steps

1. Open Power BI Desktop.
2. Select `Get data` -> `Text/CSV`.
3. Load:

```text
dashboard/car_company_dashboard_data.csv
```

4. Build the visuals from:

```text
docs/visualization_guide_powerbi_tableau.md
```

5. Save the file as:

```text
Car_Company_Database_Dashboard.pbix
```

6. Upload the `.pbix` file to the `dashboard/` folder in GitHub.

## Recommended Dashboard Pages

- Executive Sales Overview
- Product and Inventory Performance
- Customer and Payment Analysis
- Office and Employee Performance
